## 一、选品策略

### 1.1 数据来源

| 渠道 | 用途 | 优先级 |
|------|------|--------|
| **抖音商机中心** | 热词+爆品+类目机会 | ⭐⭐⭐⭐ |
| **小红书** | 审美流/颜值爆款发现 | ⭐⭐⭐⭐ |
| Amazon/TikTok/Etsy | 逆向流/海外蓝海 | ⭐⭐⭐ |
| 蝉妈妈/考古加 | 全网热销榜单 | ⭐⭐⭐ |
| 1688 热销榜 | 供应链验证 | ⭐⭐ |
| 竞品店铺 | 跟款参考 | ⭐⭐ |

> 7种选品流统一由 `clawshop-inspiration` Skill 执行。

### 1.2 选品指标

**必看指标**：
- 搜索量 > 1万/月
- 成交增速 > 20%
- 1688采购价 < 抖店售价 × 0.5（保证毛利）

**加分项**：
- 平台扶持标签（搜索扶持、上新扶持）
- 轻小件（物流成本低）
- 非标品（价格不透明）

### 1.3 利润计算公式

```
毛利 = 抖店售价 - 1688采购价 - 物流费 - 平台扣点

平台扣点 ≈ 售价 × 2%~5%（看类目）
物流费 ≈ 3~8元（看重量和距离）

目标毛利率 > 30%
```

### 1.4 避坑清单

- ❌ 品牌商品（侵权风险）
- ❌ 食品/化妆品（资质要求高）
- ❌ 超大件（物流成本吃利润）
- ❌ 价格战红海品（手机壳、数据线）
- ❌ 商品标题/描述含「代发」字样（终端消费者会看到，体验极差）
- ❌ 1688上架时间 < 2026-01-01（过时旧品，滞销风险高）

## 二、商品管理

### 2.1 标题优化

**公式**: 核心词 + 属性词 + 场景词 + 长尾词

```
✅ 好标题: 创意十二生肖黄铜钥匙扣男女情侣款汽车挂件生日礼物
❌ 差标题: 钥匙扣
```

### 2.2 质量分优化

| 分数 | 等级 | 行动 |
|------|------|------|
| 90+ | 优秀 | 保持 |
| 80-89 | 良好 | 优化属性 |
| 70-79 | 及格 | 重点优化 |
| <70 | 待优化 | 考虑下架重发 |

**优化方向**：
- 补全商品属性
- 优化主图（白底图、场景图）
- 完善详情页
- 设置合理运费模板

### 2.3 定价策略

```
建议售价 = 1688采购价 × 2.5 ~ 3.5

低价引流款: × 2.0（破0用）
常规款: × 2.5~3.0
利润款: × 3.5+
```

## 三、订单与发货

### 3.1 订单处理流程

```
抖店新订单 → 1688-shopkeeper 自动搜品匹配 → 铺货提交 → 等待发货
```

> 搜品铺货已由 `1688-shopkeeper` Skill + 每日 cron（11:30）自动完成。

### 3.2 发货时效

- 承诺发货时间：48小时
- 实际目标：24小时内
- 春节等特殊时期：设置延迟发货

### 3.3 物流选择

优先顺序：
1. 1688供应商默认物流（省心）
2. 菜鸟/极兔（性价比）
3. 顺丰（高客单价商品）

## 四、流量运营

### 4.1 免费流量

- **商品卡流量**：优化标题、主图、属性
- **搜索流量**：关键词布局
- **推荐流量**：提升点击率和转化率

### 4.2 付费推广

- **千川推广**：新品起量
- **精选联盟**：达人带货（开店满60天后可开通）

### 4.3 内容运营

- 短视频：商品展示、使用场景
- 直播：暂不开展（人力要求高）
